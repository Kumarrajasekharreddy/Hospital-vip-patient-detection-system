# Apollo VIP Patient Detection & Alert System

## Project Overview

The Apollo VIP Patient Detection System is a real-time AI-based facial recognition solution developed for Apollo Hospitals Financial District.

This system automatically:

- Fetches daily patient appointments from Apollo API
- Identifies VIP patients using EHIS database
- Detects patient faces using AI facial recognition
- Sends WhatsApp alerts when VIP patients are detected
- Stores VIP detection logs in SQL Server database
- Displays live camera feed with VIP highlighting
- Tracks waiting area patient presence in real-time
